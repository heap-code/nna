# Commands

## Global

### Lint & format

```bash
npm run lint
```

```bash
npm run lint:fix
```

### Tests

To run the tests:

```bash
npm run test
```

To run the `e2e` tests:

```bash
npm run test:e2e
```

### Other commands

Not yet documented commands:

```bash
npm run i18n:extract
```

```bash
npx mikro-orm migration:create --name <name>
```
